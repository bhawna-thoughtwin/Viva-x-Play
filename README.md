# Vivaxplay

An online betting platform built with React and Vite, offering sports betting, casino games, live dealers, and promotions.

## Features

- Sports Betting
- Casino Games
- Live Dealers
- Promotions
- User Authentication
- Policies & Terms

## Tech Stack

- React 18+
- Vite
- Tailwind CSS
- ESLint

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/       # Reusable React components
├── pages/           # Page components
├── assets/          # Images and icons
├── context/         # State management
└── App.jsx          # Main component
```
